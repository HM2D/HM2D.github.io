Projects

   Graphics Engine

A graphics engine made in OpenGL(C++) and GLSL from scratch. Inludes Multiple post processing effects such as Colour Grading, Blurring, Rendering scene from diffrent prespective, Edge Detection (Sobel Filter). Parallel loading material. Maps generated from 3 different heightmaps which are animated and cycled through. Lights and shadows from shadow maps. HUD that renders text. Texture blending also used for making the map have diffrent texture of grass and rock. Water tesselation also with diffrent heightmap. Simple and skeleton animations.






    Game Engine

A fully functional Game Engine (C++) that includes systems such as Physics, Collisions, pathfinding, networking, graphics, state machines. The game that was created using this engine is untitled goose game which is kind of a fan remake of the original game. The game is also complete in terms of 1 level which the goose has to collect all the apples and bonus items in 3 minutes of time. If goose tries to carry a bonus item the AI will try to catch him and if so the bonus item and the ai will return to their original location. The networking game follows the same behaviour for 2 players competing for the same objectives.






    Bakhtegan 1423

Survival 3D hardcore game made with Unity engine. The game takes place in Bakhtegan Area, Iran Fars. The main character "Bardia" has to eliminate the threats that have been created by the locals to destroy the natural enviroment of the area. The game includes 7 levels with 7 different maps that can be roamed freely by the player. The trailer only shows the early stages of the progress. I made this game in full but had to leave the country for my studies in the time being and wasn't able to be there for the release. The game is the product of Datis Pars company with me being the head of development for this project.






    Sadreneshin

A mobile game made with Unity engine. The game includes 7 different mini games that have the theme of balls. All games are record based and the objective is that players with top 3 high scores of the day get rewarded. This is also for the end of the month. Each day one of the 7 games is the game of the day which the records are recorded and players are awarded for playing that perticular game. This is the products of Datis Pars Co & Appsaz. I was the creator of the 7 games the head developer for Datis Pars Co & Appsaz was in charge of server side. Google play Link: Sadrneshin
